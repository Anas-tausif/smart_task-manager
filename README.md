# Smart Task Manager 🚀

Smart Task Manager is a full-stack web application built using:

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js + Express  
- Database: MySQL  
- Authentication: JWT  

The application allows users to register, login, and manage tasks with full CRUD functionality (Create, Read, Update, Delete).


## Features

- User Registration & Login
- Add, Edit, Delete Tasks
- Change Task Status (Pending / Completed)
- Dashboard Task Summary
- Reports (Chart Visualization)
- Profile Update & Password Change
- Responsive Design


## How to Run the Project

### 1️⃣ Install Dependencies

Open terminal inside the project folder:



npm install




### 2️⃣ Create `.env` File

Create a `.env` file in the root directory:

PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=smart_task_manager
JWT_SECRET=your_secret_key




### 3️⃣ Start the Server
npm run dev 

or 

npm run dev




## Web Link (Local)

After starting the server, open:
http://localhost:5000


The application will run in your browser on this link.



## Database

Make sure MySQL is running and the database `smart_task_manager` is created before starting the server.
